# TypeScript Standards

Always use strict typing.

Rules:

- Avoid "any"
- Prefer explicit interfaces
- Use type-safe API responses
- Extract types to /types when reused

Naming:

- Interfaces: PascalCase
- Types: PascalCase
- Variables: camelCase
